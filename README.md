# codex-token-monitor

Мониторинг и аудит использования токенов Codex. Сервер отслеживает расход токенов в реальном времени, ведёт историю по проектам и рассчитывает стоимость на основе заданных тарифов.

## Структура

- `config/` — конфигурация проектов и цены на токены
- `scripts/` — скрипты сервера и аудита
- `static/` — веб-интерфейс (фронтенд)
- `start_codex_token_monitor.bat` — запуск сервера

## Запуск

Запустите `start_codex_token_monitor.bat` — сервер будет доступен по адресу http://127.0.0.1:8765
