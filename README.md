# Песчаный рубеж

Небольшая автономная браузерная RTS в оригинальном пустынном sci-fi сеттинге. Игра работает на чистых HTML, CSS и JavaScript без сборщика, сервера, библиотек, CDN и внешних ассетов.

## Запуск

1. Откройте папку `desert-rts`.
2. Дважды щёлкните `index.html` или откройте его через браузер.
3. Нажмите «Начать операцию».

## Цель

Сохранить командный узел и отразить пять волн противника.

## Управление

- ЛКМ — выбрать юнит или сооружение; разместить выбранное строение.
- ПКМ — приказать выбранному юниту двигаться или атаковать цель.
- `1` — режим строительства экстрактора.
- `2` — режим строительства энергоблока.
- `3` — режим строительства завода.
- `4` — режим строительства турели.
- `Esc` — отменить строительство.
- `Пробел` — пауза / продолжить.

## Игровые системы

- Светокамень — основной ресурс.
- Экстрактор позволяет создавать буровой дрон, который автоматически добывает ресурс.
- Энергоблок увеличивает доступную мощность.
- Завод создаёт разведчиков и штурмовые шагоходы.
- Турели автоматически атакуют противника при достаточном энергоснабжении.
- Вражеские волны становятся многочисленнее и получают тяжёлые машины.

## Совместимость

Нужен современный браузер с поддержкой Canvas 2D и ES6: актуальные Chrome, Edge, Firefox или Safari.
